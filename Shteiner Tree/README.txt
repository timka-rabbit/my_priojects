This application demonstrates the work of some algorithms for constructing a Steiner tree.

To run the application through Visual Studio open the file CristalProc.sln.

Go to the folder "Examples of Results" to view some screenshots of the application.

Libraries:
"AlgorithmClass" contains realization of Kruskal, Brute force and AntColony algorithms.
"MathClass" contains randomizer and distance search methods.
"PointClass" contains realization of point class.
"Graphic UI" contains interface realization and run algorithms.
